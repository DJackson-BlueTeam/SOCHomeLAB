# Complete SOC Lab Procedure: Windows Server & Active Directory Setup

## Phase 1: Virtual Box VM Creation

**1. Create New VM**

**2. Name & OS**
-  Name: SOC-DC01 (What is desirable)
- Type: `Microsoft Windows`` | Version: Windows 2022 (64-bit)` (or latest version)

**3. The "Unattended" Fix (Crucial): Uncheck the box "Skip Unattended Installation"**
- This prevents VirtualBox from trying to automate the install, which causes the "License Terms" error.

**4. Hardware**
- RAM: At least 2048 MB (4096 MB is better if your host PC allows).
-  Processors: 2 CPUs

**5. Hard Disk**
    - Create a virtual disk of at least 50 GB (Desktop Experience needs more space than core)
