# FLARE-VM: The Installation Procedure 

This guide provides end-toend process for building a FLARE-VM malware analysis lab. The Host Machine of the VM OS must have a minimum of 16 GB RAM and an additional up to 500 GB to 1 TB of SSD.
https://www.youtube.com/watch?v=wYBgAb0rw1k <--- Basic VM/Windows Installion

Phase 1: Initial Windows VM Creation
---
**1. Download Windows ISO**
- Obtain a Windows 10 Pro or 11 ISO (If downloaded windows 10 home, it can be updated virtually shown in the later phase.)
https://www.microsoft.com/en-us/software-download/windows10?msockid=14411f7e24d963171a8309ea258b6220

**2. VirtualBox Settings**
- Name: (A name tailored to Malware Lab)
- Base Memory: At least 4096 MB (4GB)
- Processor: At least 2 CPUs.
- Hard Disk: Create a Virtual Hard Disk of 100 GB (Dynamically Allocated).

**3. Installing Windows**
- Walk through a standard windows installing. When prompted for a user, use a simple name `Malware Lab` or `SOC Lab` or anything in that aspect.
